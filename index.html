What is HTML Minification?

HTML minification is the process of reducing the size of HTML files by removing unnecessary characters without changing the functionality of the code. This includes removing comments, white spaces, and optimizing the markup structure.

Before and After Example

Original HTML

<!DOCTYPE html>
<html>
    <head>
        <title>My Page</title>
    </head>
    <body>
        <div class="container">
            <h1>Hello World</h1>
            <!-- Navigation menu -->
            <nav>
                <ul>
                    <li>Home</li>
                </ul>
            </nav>
        </div>
    </body>
</html>

Minified HTML

<!DOCTYPE html><html><head><title>My Page</title></head><body><div class="container"><h1>Hello World</h1><nav><ul><li>Home</li></ul></nav></div></body></html>

Benefits of Minification

  • Reduced file size and faster loading times
  • Lower bandwidth usage
  • Improved website performance
  • Better user experience
  • Lower hosting costs

Common Minification Techniques

  • Remove whitespace, line breaks, and indentation
  • Remove HTML comments
  • Remove unnecessary quotes from attributes
  • Remove default attributes
  • Shorten color values
  • Remove empty attributes

Popular Minification Tools

  • HTMLMinifier
  • Grunt-contrib-htmlmin
  • Gulp-htmlmin
  • webpack with html-minimizer-webpack-plugin
